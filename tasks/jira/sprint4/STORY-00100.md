# STORY-00100: Cutout Illustrated 插画精修

**Epic**: E-005 UX Polish
**Sprint**: Sprint 4
**Points**: 5
**Owner**: Frontend
**Status**: Done

## Description
As a first-time iPhone user, I want the illustrations and icons to feel hand-crafted with visible torn-paper edges, so the app matches the promised Cutout Illustrated aesthetic instead of looking like emoji placeholders.

## Expected Result
Home 页 HeadphoneListener 头像、Learn/Review/Library 三张卡片图标、彩色条纹分隔线全部具备：(a) 可见撕纸边缘 (b) 多层叠加阴影 (c) 手工错位/不完美对齐。视觉与 Sprint 0 CP1 确认的 Style F 参考图一致。

## Acceptance Criteria
- [ ] `tearing.strong.scale` 从 9 提升到 ≥ 15，`baseFrequency` 微调让边缘噪声粒度更粗（更像真实撕纸）
- [ ] HeadphoneListener 增加至少 2 层背景阴影层（错位 3-5px 的深棕/暗红副本）
- [ ] Learn/Review/Library 三个图标风格统一：全部使用 TearingDefs + 3 层色（底/主/亮点）
- [ ] WovenDivider 从等宽 Rect 拼接改为不规则宽度（每条 8-16px 随机）+ 上下边缘用撕纸 filter
- [ ] iPhone 14 视口下截图与旧图并排对比，撕纸感肉眼可辨（非放大 200% 才能看清）
- [ ] 三档视口（iPhone SE / 14 / 15 Pro Max）截图均无渲染退化
