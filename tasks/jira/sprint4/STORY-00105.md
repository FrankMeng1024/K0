# STORY-00105: 微交互全套 — tilt / haptic / 页面过渡

**Epic**: E-005 UX Polish
**Sprint**: Sprint 4
**Points**: 3
**Owner**: Frontend
**Status**: Done

## Description
撕纸风情绪核心是 tactile（触感），当前全应用无微交互——按下卡片无 tilt/scale、无 haptic 视觉暗示、页面切换无撕纸翻面感。UX Sprint 3 review 指出"微交互缺失是让整体感觉粗糙的重要原因之一"。

## Acceptance Criteria
- [ ] Home 3 张入口卡片和 GoalSelect 5 个按钮：按下时 `scale: 0.97` + `rotate: 0.5deg`（撕纸翻起微感），松开回弹
- [ ] Home PasteBar 按钮按下时叠加轻微阴影位移 2px
- [ ] Learn 页"开始"按钮、GoalSelect 目标按钮、Episode 步骤按钮：按下时 Haptics.impactAsync (Light) —— Web 平台 no-op（已有 try/catch）
- [ ] Expo Router 全局配置页面过渡动画：`animation: 'slide_from_right'` 改为 `animation: 'fade_from_bottom'`（撕纸浮起感）
- [ ] Episode 页 pack 加载完成时，snapshot 卡片从下方浮入（Animated.spring 弹性）
- [ ] 三档视口截图（静态画面无法完全验证动效，QA 需拍视频或多帧序列）
