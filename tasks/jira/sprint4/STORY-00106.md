# STORY-00106: iOS 原生返回样式 + back gesture

**Epic**: E-005 UX Polish
**Sprint**: Sprint 4
**Points**: 1
**Owner**: Frontend
**Status**: Done

## Description
当前"← 首页" / "← 返回" 使用中文文字+ASCII 箭头，非 iOS 原生风格。原生 iOS 返回是"‹ 上一屏名"或大标题+侧边 back gesture。

## Acceptance Criteria
- [ ] 所有返回按钮从 `<Pressable>← 首页</Pressable>` 改为使用 `‹` 字符 + 上一屏名（如 `‹ 首页` / `‹ 选目标`）
- [ ] 字号 fontFamily 用 iOS system font（fonts.ui 已定义），大小 17pt
- [ ] 侧边 swipe back gesture 生效（Expo Router 默认开启，验证不被 disable）
- [ ] 三档视口截图确认
