# STORY-00107: pill/chip 视觉分级

**Epic**: E-005 UX Polish
**Sprint**: Sprint 4
**Points**: 2
**Owner**: Frontend
**Status**: Done

## Description
BubbleTag 组件当前一套样式用在三种角色上：导航按钮（Home 卡片 tag）、状态标签（GoalSelect 右上、Episode 顶部 pill）、可操作 chip（Episode 时长/受众 tag）。用户无法判断哪个能点。

## Acceptance Criteria
- [ ] BubbleTag 组件增加 `variant` prop: `'nav' | 'status' | 'action'`
- [ ] nav 变体：现有样式（白底 + 前置红点）保持
- [ ] status 变体：无边框、透明背景、字号缩小 2pt、颜色变深棕
- [ ] action 变体：撕纸边框 + 主题色底 + 无红点
- [ ] Home 卡片 tag 用 `nav`；Episode 顶部当前状态 pill 用 `status`；Episode 时长/受众 tag 用 `action`
- [ ] GoalSelect 右上 pill 已在 STORY-00102 移除，此处不再涉及
- [ ] 三档视口截图对照
