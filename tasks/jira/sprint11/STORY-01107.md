# STORY-01107: Prompt v2 (Step 1 + Step 2 独立 prompt)

**Epic**: E-002/E-003 · **Sprint**: 11 · **Points**: 2 · **Owner**: Backend · **Status**: Todo

## Description
拆 `backend/prompts/generate-pack.zh.md` 为两个独立 prompt：
- `snapshot.zh.md`：Step 1，输入全转录，输出 snapshot (含 audience + 动态 worthListening)
- `pack.zh.md`：Step 2，输入 Step 1 worthListening 段落 + mode，输出学习包

## Step 1 输出 JSON schema
```json
{
  "oneSentence": "≤25字",
  "audience": ["产品经理", "创业者"],
  "valueScore": { "density": 8, "novelty": 7, "actionability": 6 },
  "estimatedCostMinutes": 10,
  "worthListening": [
    { "startSec": 300, "endSec": 420, "reason": "为什么值得听（一句话说明）", "quoteParagraph": "原文段落 150-400 字" }
  ],
  "skippable": [
    { "startSec": 0, "endSec": 90, "reason": "广告" }
  ]
}
```

**观点入选标准**（Step 1 prompt 硬约束）：一段满足 ≥2 条入选 worthListening
- 信息密度：一句话含 2+ 个非常识事实
- 反常识度：挑战默认认知
- 可行动性：听完能立刻做/试/避
- 可迁移性：方法/框架能用到别的场景
- 原创性：嘉宾自己想的

**动态密度**：短(<30min) 2-3 段 / 中(30-60min) 3-5 段 / 长(60-120min) 5-8 段 / 极长(120+) 8-12 段

## Step 2 输出 JSON schema (8 字段卡片)
```json
{
  "steps": [
    { "title": "背景理解", "content": "...", "timestamp": 30 },
    // 6 步
  ],
  "concepts": [
    { "term": "...", "plain": "小白解释", "context": { "text": "原文语境", "timestamp": 123 }, "related": "延伸" }
  ],
  "cards": [
    {
      "title": "观点陈述",
      "type": "opinion|method|case|reflection|action",
      "core": "3-5 句核心解释",
      "context": "原文语境+时间戳",
      "usage": "1-2 个用例",
      "challenge": "反面视角",
      "source": { "quote": "嘉宾原话摘录", "timestamp": 123 },
      "myApplication": "AI 建议"
    }
  ],
  "actions": {
    "today": "今天 1 件事",
    "week": "本周 1 件事",
    "longterm": "值得研究 1 问题"
  }
}
```

**mode 分支**：
- quick: cards 3-5 张（挑最核心观点）
- deep: cards 3-18 张（覆盖所有核心观点，动态密度：短 3-6 / 中 6-12 / 长 10-18）

**输出语言**：所有输出都用中文（CR-012）
