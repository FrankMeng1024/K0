# STORY-01006: 闪卡模式 (Flashcard Mode)

**Epic**: E-005 Review System
**Sprint**: Sprint 10
**Sprint Goal**: PRD Must-Have 收尾
**Points**: 2
**Owner**: Frontend
**Status**: Done (由 Sprint 8 Loop 30 已实装 — Review 屏本身就是闪卡模式)

## Description
As a user reviewing cards, I want a flashcard mode: front shows a question/prompt (e.g. "什么是 BM 学徒？"), back shows the answer + source timestamp. Tap to flip.

## Expected Result
- Review 屏"复习节奏"页面进入闪卡模式
- 全屏一张卡片，先显示 question
- 点击/滑动翻面显示 answer
- 记得/模糊/不记得 3 按钮反馈 SRS
