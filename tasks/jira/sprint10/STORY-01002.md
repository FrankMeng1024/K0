# STORY-01002: 卡片删除 UI (Card Archive Button)

**Epic**: E-003 Learning Pack
**Sprint**: Sprint 10
**Sprint Goal**: PRD Must-Have 收尾
**Points**: 1
**Owner**: Frontend
**Status**: Done

## Description
As a user, when I see a knowledge card that isn't useful to me, I want to remove it (soft delete via archived=true), so my Library isn't polluted.

## Expected Result
- 每张知识卡片右上角除 ★ 收藏外，新增 🗑 删除按钮
- 点击后 confirm 一次 → PATCH archived=true → 卡片本地淡出移除
- Library 屏只显示未 archived 的卡片
- Review 队列只针对 favorited=true AND archived=false 的卡片
