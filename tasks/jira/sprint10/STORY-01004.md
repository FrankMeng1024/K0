# STORY-01004: 行动清单勾选进 Review (Action → Review Queue)

**Epic**: E-003 Learning Pack + E-005 Review
**Sprint**: Sprint 10
**Sprint Goal**: PRD Must-Have 收尾
**Points**: 2
**Owner**: Backend + Frontend
**Status**: Done

## Description
As a user, when I check an action item in the action plan (今天/本周/长期), I want it to enter my Review queue as a "commitment" I need to follow up on, so I don't just read and forget.

## Expected Result
- 行动清单每条前加 checkbox
- 勾选后进入 Review 队列（等同一张"行动卡"）
- Review 屏可看到"你承诺的行动"分类
- 完成后可标记 done
