# STORY-01001: 概念解释器 (Concept Explainer)

**Epic**: E-003 Learning Pack
**Sprint**: Sprint 10
**Sprint Goal**: PRD Must-Have 收尾
**Points**: 3
**Owner**: Backend + Frontend
**Status**: Done

## Description
As a user, when I see unfamiliar terms in a learning pack (行业术语、人物、公司、方法框架), I want to tap them and get a three-layer explanation (小白解释 / 原文语境 / 延伸理解), so I don't get stuck on jargon.

## Expected Result
- Episode 页展示的每个"关键概念"是可点开的 chip / underlined phrase
- 点击后弹出/展开三层解释
- 中英文播客都能生成对应语言的概念

## Acceptance Criteria
- [x] Backend: GLM prompt 加"提取 5-8 个关键概念，每个 3 层解释 { plain, context: {text, timestamp}, related }"（backend/prompts/generate-pack.zh.md + .en.md 新增 concepts schema）
- [x] Backend: pack.concepts 持久化 — 采用 pack_json 内嵌方式（learning_packs.pack_json JSON 已有，无需新增列；结构上更整洁）
- [x] Frontend: Episode 页新增"关键概念"折叠面板（app/episode/[id].tsx ConceptsPanel 组件，line 472+）
- [x] Frontend: 每个概念标题可点击展开三层（plain / context / related）
- [x] Frontend: concepts 数组为空/undefined 时整个 section 隐藏（parsePack 兜底 `Array.isArray(raw?.concepts) ? raw.concepts : []`）
