# STORY-00003: RN 3 入口首页（风格 F 移植 —— 更细致抽象）

**Epic**: E-004 Learning Player / E-007 Foundation（CR-001 后重写）
**Sprint**: Sprint 1
**Points**: 5
**Owner**: Frontend
**Status**: Done

## Description
As Frontend, I want to 用 React Native + expo-router + NativeWind 实现 Home 三入口页面（Learn/Review/Library），风格严格对齐 `style-demos/k0-demo-f-cutout-illustrated.html` 的 Home 页，但按用户 CP1 附加要求"更细致 抽象一些"重新做 SVG 插画。

## Expected Result
- `app/index.tsx` = Home 屏幕
- `app/learn.tsx` = Learn 空壳（Sprint 2 补内容）
- `app/review.tsx` = Review 空壳
- `app/library.tsx` = Library 空壳
- 4 屏幕都用 Expo Router file-based routing
- Home 页视觉严格对齐风格 F：
  - 大 Bagel Fat One 胖字母 "Listen. / Learn."
  - 耳机人物侧脸 SVG 剪影（用 react-native-svg 重写，比 demo F 更抽象几何化）
  - 3 张撕纸卡片：Learn（红）/ Review（黄）/ Library（蓝）
  - 每张卡片有主视觉 SVG（手/大脑/书架），比 demo F 版更精致
  - 织布条分隔（用 SVG Pattern）
  - 白色椭圆气泡标签
  - 4 边有 padding ≥ 40pt 上下 / 24pt 左右
