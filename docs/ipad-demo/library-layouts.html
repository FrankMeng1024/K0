K0 iPad「知识收藏页 Library」布局方案（iPad Pro 11" 横屏 1194×834，页面按 68% 缩放展示）

收藏页有「学习包 / 卡片」两个 tab、mode filter（全部/精学/速学/跳过）、收藏 filter、知识图谱入口。手机版是单列列表，iPad 横屏把列表变多列网格。用真实内容占位。emoji 只是占位（真机是撕纸插画）。选一个方向。

方案 A — 左侧固定 filter 栏 + 右主区多列网格
左窄栏常驻所有筛选（tab 切换 + mode + 收藏 + 图谱入口），右主区三列卡片网格。优点：筛选一直可见、切换零跳动、和学习包页「左栏」骨架一致；缺点：左栏占宽，网格三列。

方案 B — 顶部 tab + filter 横排 + 全宽四列网格
保留现有上下结构，把 tab/filter 放顶部横排，下方全宽四列网格（比手机单列多一倍密度）。这里展示「卡片」tab（收藏 filter 已选）。优点：改动最小、最接近现有手机版放大、网格四列信息密度最高；缺点：滚动时筛选会滚走（不像 A 常驻）。
